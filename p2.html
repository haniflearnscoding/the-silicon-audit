Screen 04 / 04   Mobile View
Browser Screenshot · iPhone 17 · 390px viewport Responsive Layout — Single Column
the-silicon-audit · Mobile
Single-Column Feed
Both sidebars hidden at ≤768px. Full-width cards stack vertically. The feed is the entire experience on mobile.
SVG Scales Intact
Gradient bars use width:100% — the proportional encoding scales correctly to any viewport. Large stat numbers (+340%, −91%) remain readable at mobile scale.
Stamp Clearance
Quote text gets right padding on mobile so the rotated RETRACTED stamp does not overlap the CEO quote. Stamp size reduces proportionally.
01
Gradient Bar Encoding
Each card uses RGB saturated linearGradients — bar width (or height) is proportional to value. The empty space after the actual bar IS the gap: void as data, following the IIB "ink = value" principle.
02
Per-Card Gradient Identity
Each company gets a distinct gradient pair: cyan→blue (OpenAI), purple→indigo (Meta), red→dark (Tesla), teal→cyan (Google expected), amber→orange (Microsoft claimed). Color is strictly semantic.
03
Mixed Visualization Grammar
Cards 1–2 use horizontal proportional bars. Card 3 uses a calendar unit grid. Card 4 uses a horizontal bar with error labels. Card 5 uses vertical bars. Each card's form is chosen for the claim type.
