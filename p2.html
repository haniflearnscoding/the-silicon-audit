Area-Accurate Encoding
Unlike bar charts that encode length, the visualizations use proportional area — following Information is Beautiful's principle that ink should map accurately to value, making magnitude viscerally legible.

Drawing on Dear Data's unit-mark vocabulary, each dot and symbol represents a discrete real-world event — a promise made, an error documented, a deadline missed.
